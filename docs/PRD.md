# PRD — Aplikasi Kantin SD / SD Islam

## Tujuan Produk
Membantu penjaga kantin mencatat penjualan dan stok harian dengan mudah, serta membantu yayasan memantau laporan penjualan secara rapi, transparan, dan konsisten.

## Target Pengguna
- Penjaga Kantin (user harian, non-teknis)
- Yayasan / Admin (monitoring & laporan)

## Masalah Utama
- Pencatatan penjualan tidak rapi
- Stok sering tidak terpantau
- Laporan harus ditanya manual

## Ruang Lingkup (MVP v1)
- Login berbasis role
- Master barang (nama, harga, stok)
- Input penjualan cepat
- Stok otomatis
- Tutup penjualan harian
- Laporan harian, mingguan, bulanan
- Dashboard ringkas

## Di Luar Scope (v1)
- Pembayaran digital
- Hutang siswa
- Promo / diskon
- Multi sekolah

## App Flow
Penjaga:
Login → Dashboard → Input Penjualan → Tutup Hari

Yayasan:
Login → Dashboard → Pilih Periode → Lihat Laporan
